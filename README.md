## 🚀 快速开始 (Getting Started)

### iOS 开发环境

确保你的 Mac 安装了 **Xcode 15+** 并且系统在 **macOS Sonoma** 以上。

1. 克隆仓库：
   ```bash
   git clone https://github.com/vivacious1024/MindLog.git
   ```
2. 打开项目：
   双击 `MindLog_.xcodeproj`。
3. 配置签名：
   在 Xcode 的 `Signing & Capabilities` 中选择你的开发者账号。
4. 运行：
   选择模拟器 (iPhone 15 Pro Max 推荐) 并按下 `Cmd + R`。
